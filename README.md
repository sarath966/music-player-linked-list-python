# Music Player using Circular Doubly Linked List (Python)

This project implements a console-based music player using a circular doubly linked list in Python.

Features:
- Play songs
- Pause and Resume
- Next song
- Previous song
- Insert song at position
- Delete song from playlist
- Display playlist

Technologies Used:
- Python
- Pygame
- Circular Doubly Linked List
